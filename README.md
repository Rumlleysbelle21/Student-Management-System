# Student Management System (C++)

## 📌 Description
This is a simple console-based Student Management System built in C++. It performs basic CRUD operations (Create, Read, Update, Delete).

## 🚀 Features
- Add Student
- View Students
- Search Student by ID
- Delete Student

## 🛠 Technologies Used
- C++
- STL (Vector)
- Console Application

## ▶️ How to Run
1. Compile the code:
   g++ student.cpp -o student

2. Run the program:
   student.exe

## 🎯 Purpose
This project is created for learning basic C++ concepts like functions, loops, structures, and vectors.
